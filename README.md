### KBS미디어 — 개발 PL / iOS 리드 개발자
**2022.05 ~ 재직 중 (약 4년)**
KBS+ / KBS My K · KBS 공식 OTT·실시간 스트리밍 iOS 앱 (App Store 운영 중)  
※ 네이티브 전면 재구축(KBS+) 2023.03 착수  
**포트폴리오** : [KBS+ 포트폴리오](./portfolio/KBSPlus_Portfolio.md)

레거시 Objective-C/JavaScript 브릿지 앱의 Swift 클린 아키텍처 네이티브 전환을 단독 수행. 아키텍처 설계부터 릴리스 운영까지 전 과정 담당. 월 활성 사용자(MAU) 약 40만 명 규모의 라이브 서비스를 운영 중단 없이 전환.

- **레거시 앱의 무중단 네이티브 전환.**  
  Objective-C/JS 브릿지 구조를 화면 단위로 점진 마이그레이션하여, 서비스 중단 없이 3년간 100% Swift로 전환 완료.
- **자체 네트워킹 레이어(PlusNetwork) 설계.**  
  서드파티 네트워킹 라이브러리에 의존하지 않고 URLSession 기반으로 직접 구축. 엔드포인트를 값 타입으로 선언하고 요청을 use-case 단위로 캡슐화하는 타입 안전 구조로, 산발적 네트워크 호출을 일관된 패턴으로 통일하여 신규 API 추가 비용 및 응답 모델 오류 감소.
- **서버 주도 UI(SDUI) 도입 주도.**  
  홈 화면을 서버 응답으로 구성하는 컴포넌트 기반 구조로 전면 재설계. 서버가 컴포넌트 타입·노출 여부·데이터 소스·노출 시작/종료 시각을 내려주면 클라이언트는 이를 타입 안전한 enum으로 매핑해 렌더링. 앱 배포·심사 없이 홈 구성과 편성을 서버에서 제어할 수 있어, 방송사 특성상 잦은 편성 변경·특보를 즉시 반영. 다중 컴포넌트 동적 배치, 개인화 섹션, 컴포넌트 단위 테스트까지 포함. 스키마 설계부터 기획·백엔드 연동 방식까지 주도.
- **네트워킹 동시성 모델 점진 개선.**  
  completion 클로저 기반에서 출발해 Combine Publisher, async/await를 단계적으로 도입하며 요청 흐름을 정리. 요청마다 URLSession을 생성하던 초기 구조를 공용 세션(URLSession.shared) 재사용으로 전환하여 자원 누수 위험 제거. 운영 중인 서비스를 유지하면서 동시성 모델을 세대 교체.
- **UI 구성 방식 점진 전환.**  
  XIB 기반에서 출발해 '화면 단위 1 스토리보드' 규칙으로 정리하여 화면별 책임 분리와 생산성 확보. 이후 스토리보드의 머지 충돌·재사용성·동적 레이아웃 한계를 인식하고, 신규 및 재작성 화면부터 코드 기반 UI로 전환 진행 중.
- **iPad 적응형 레이아웃 및 iOS 26 신규 디자인 대응.**  
  회전·창 크기에 따라 세로 스택과 좌우 분할을 동적 전환하는 레이아웃 시스템 구축. iOS 26 내비게이션 appearance 변경에 대응하여 내비게이션 바 재설계.
- **의존성 분리 및 주입 구조 도입.**  
  네트워크 transport 계층과 Service 계층을 프로토콜 기반으로 추상화·주입 가능하도록 리팩터링하여 모듈 간 결합도를 낮추고, 구현체 교체·검증이 용이한 구조 확보.
- **협업 기반 공용 모듈 설계 및 제공.**  
  리드로서 네트워킹·공통 UI 컴포넌트·코드 컨벤션 등 공용 기반을 직접 설계·제공하여, 신규 인원이나 외주 인력이 합류해도 동일한 구조 위에서 바로 작업할 수 있는 환경 구축. 개인 의존도를 낮추고 협업 진입 비용 최소화.
- **국제 스포츠 중계 대응(월드컵·올림픽 등).**  
  IOC·FIFA 등 국제 기구가 요구하는 앱 보안 절차 및 장애 발생 시 대응 체계를 협의·이행. 트래픽이 집중되는 대형 국제 행사 기간 동안 앱 클라이언트의 안정적 재생을 지원.
- **공공·규제 행정 대응.**  
  개인정보 영향평가, 방송통신위원회 관련 행정 절차, 보안성 검토 등 공기업·방송 규제 환경에서 요구되는 행정 업무를 개발과 병행하여 직접 처리.
- **기타 기능 구현.**  
  VoiceOver 접근성, 실시간 채팅, Google Cast, PIP, 유니버설 링크 딥링킹, Apple 로그인, 간편모드·키즈모드.

신규 Swift 코드베이스 전반(네트워킹·플레이어·UI·도메인 등 핵심 모듈) 설계·구현.
